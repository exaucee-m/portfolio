Skills

HTML

The Hyper-Text Markup Language is used to structure websites. Able to confidently use this technology to create websites as well as understanding how to use various tags such as headers, anchors, images, tables, and forms to name a few.

CSS

Cascading Style Sheets is a language which focuses on the styling elements of a webpage. From the use of colours, sizing, positioning, the box model and display properties it ensures the pages are designed as desired. Confident with both inline and external CSS, syntax and debugging tools.

JavaScript

JavaScript is a scripting language used alongside HTML and CSS. It allows functionality of the webpages which enables users to experience an interactive website, for example, using a search box or a reloading a new page after submitting a form. Confidently able to use data types; strings, Boolean, as well as functions and alerts in order to produce working websites.

Bootstrap

Bootstrap is a popular framework used to develop responsive websites across a range of devices. This front-end focused framework uses pre-built code using HTML, CSS, and JavaScript to aid with efficiency and consistency when developing websites. Confidently able to use this framework as well as understanding the various components, breakpoints, and functions.

Web Design & Marketing

Web design & Marketing are useful skills to have relating to web development. Having studied both Marketing as an undergraduate degree and a Web Design module within the Full-Stack Web Developer course, there is good knowledge on the use of colour theory, typography, user interface and experience as well as consumer behaviour and branding. Combined with practice through design tools such as Canva, enables confidence in producing well designed, easy to navigate, functional websites.

jQuery

jQuery is a popular JavaScript library that simplifies HTML document manipulation, event handling, and animation. It provides cross-browser compatibility and reduces the amount of code needed to achieve common tasks. jQuery has a large community and extensive documentation, making it easy to learn and implement.

Git, GitHub and Version Control

GitHub is a web-based platform that uses Git as its version control system. Git is a distributed version control system that allows multiple developers to work on the same codebase simultaneously, keeping track of all changes made to the code. With GitHub, developers can collaborate on projects, track changes, and review each other's work.

SQL

SQL (Structured Query Language), is a programming language used to manage and manipulate data stored in relational databases. SQL allows users to create, modify, and query databases by using a variety of commands and functions. It is a powerful tool for working with data and is essential for many fields that rely on accurate and efficient data management.
